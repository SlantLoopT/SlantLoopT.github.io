vector

std::vector 是 C++ STL 提供的动态数组容器，可以自动扩展、支持随机访问.
std::vector 是一个模板类，模板参数 T 代表元素的类型。你可以在创建 vector 时指定 T 是某种类型，T 可以是任何类型：基本数据类型、类、结构体、指针、甚至其他容器。

添加、删除元素

v.push_back(x) 添加元素到末尾
v.pop_back() 删除最后一个元素
v.insert(pos, x) 在 pos 位置插入 x
v.erase(pos) 删除 pos 位置元素
v.clear() 清空所有元素
访问元素
操作 说明
v[i] / v.at(i) 访问第 i 个元素（at() 有边界检查）
v.front() 第一个元素
v.back() 最后一个元素
v.data() 返回底层数组指针（T* 类型）
