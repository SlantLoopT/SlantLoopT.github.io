4.size_type

size_type 是专门用于表示容器大小的无符号整数类型
size函数返回的是std::size_type类型的值

5.string、字符数组与C风格字符串

字符数组 是一种数据结构（char 类型的数组），不一定是字符串，没有 \0 时它就是普通字符集合
C 风格字符串 是一种协议：以 \0 结尾的字符序列，必须以 \0 结尾，所有 C 风格字符串操作函数（如 strlen, strcpy, strcmp）都依赖这个结尾。
