数据类型

1.基本内置类型

基本内置类型分为算数类型和空类型

(1)算数类型

算数类型分为整型（包括字符和布尔）和浮点型

类型 说明 最小尺寸
short 短整型 16位
int 整型，存储整数 16位
long 长整型 32位
long long 长整型 64位
char 字符 8位
wchar_t 宽字符，用于扩展字符集 16位
char16_t Unicode字符，用于扩展字符集 16位
char32_t Unicode字符，用于扩展字符集 32位
bool 布尔类型 未定义
float 单精度浮点数 6位有效数字
double 双精度浮点数 10位有效数字
long double 扩展双精度浮点数 10位有效数字

除去布尔和扩展的字符型之外，其他整型（包括char）可划分为带符号的和无符号的。
与其他整型不同，字符型分为char、signed char、unsigned char，特别注意char与signed char不同，char是带符号还是无符号由编译器所决定。

(2)空类型

用法 说明
void func() 函数无返回值
int func(void) 函数无返回值
void* ptr 通用指针，可指向任意类型
void 不能作为变量类型 不允许 void x;

2.指针类型

int型的指针实际上是指向int型变量的指针

类型 说明
int * 指向int的指针
char * 指向字符的指针
void * 通用指针

通过*的个数可以区分指针的级别

用法

1.指向对象

2.指向nullptr（空指针，合法但不能解引用）

3.指向未初始化的内存（野指针）

4.指向已释放的对象（悬空指针）

3.引用类型

作为对象的别名

特性 说明
必须初始化 声明引用时必须立刻绑定一个变量
不能为 null 引用必须绑定到合法对象
不可更换绑定对象 引用一旦绑定，不能改为引用别的对象
是原变量的别名 所有操作都是作用于原对象

指针的引用

4.数组类型

数组声明 数组类型 说明
int arr[5]; int[5] 含有 5 个 int 的数组
char str[20]; char[20] 含有 20 个字符的数组
int matrix[3][4]; int[3][4] 包含 3 个 int[4]的数组
指针数组，指向数组的指针，引用数组，数组的引用
名称 语法 说明 解析
指针数组 int* arr[3]; ✅ 一个数组，每个元素是 int* int*[3]（数组，元素类型是 int*）
指向数组的指针 int (*p)[3]; ✅ 一个指针，指向一个 int[3] 数组 int (*)[3]（指针，指向 int[3]）
引用数组 ❌非法 int& arr[3]; ❌ 错误！不能定义引用的数组 ❌ 非法语法
数组的引用 int (&arr)[3] = someArr; ✅ 一个引用，引用整个数组 int (&)[3]

5.类类型

定义类类型

标准库中很多常用类型其实就是类类型

类型 说明
std::string 管理字符串的类
std::vector 动态数组容器
std::map<K, V> 映射表，键值对集合
std::ostream 输出流类

6.auto

定义

根据变量的初始化表达式自动推导出变量的类型。
auto 变量名 = 初始化值;

常见推导
初始化表达式 推导结果类型 说明
auto i = 10; int 整数字面量默认为 int
auto i = 3.14; double 浮点字面量默认是 double
auto i = “hello”; const char* 字符串字面量是 const char*
auto i = {1, 2, 3}; std::initializer_list 特例：纯 {} 会推成初始化列表
auto i = &x; int* 取地址操作，返回的是一个指向 int 的指针
auto i = ‘A’; char 字符字面量
auto i = true; bool 布尔值
auto i = new int(5); int* (new 表达式返回指针) new 表达式返回指针
auto i = std::vector{1,2,3}; std::vector* 大括号初始化，推导出容器类型
auto i = someFunction(); 取决于返回值类型 返回函数类型
和引用、指针配合
初始化表达式 推导结果类型 说明
auto x = expr; 拷贝expr的类型 值拷贝
auto& x = expr; &expr类型 不想拷贝时用（保留原对象）
const auto& x = expr; const &expr类型，expr可以是变量 既避免拷贝又保护不被修改
const int x = 42; auto c = x； int const 被去掉
const int x = 42; auto& c = x； const int& 显式引用，保留 const
auto* p = &expr; expr类型* expr类型的指针
auto i = &expr; expr类型* 同上

7.decltype(expr)
